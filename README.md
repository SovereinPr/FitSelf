# FitSelf - Браузерное приложение для фитнеса

Включающее в себя: Калькулятор калорий, таймер медитаций и трекер задач

## Run Locally

Clone the project

```bash
  git clone https://github.com/SovereinPr/FitSelf.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
